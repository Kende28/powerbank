powerbank.id positive intiger that auto increments.
powerbank.name string, name of the specific model of powerbank.
powerbank.brand string, brand of the specific powerbank.
powerbank.battery_time positive intiger, amount of time the powerbank can charge for in hours.
powerbank.charge_duration positive intiger, amount of time the powerbank needs to recharge in hours.
powerbank.cost not negative intiger, amount of Ft the powerbank costs.
powerbank.available boolean, if true the powerbank is available for purchase.

/addpb

name string input, value for powerbank.name. Is required.
brand string input, value for powerbank.brand. Is required.
battery_time number input, value for powerbank.battery_time. Is required.
charge_duration number input, value for powerbank.charge_duration. Is required.
cost number input, value for powerbank.cost. Is required.
available select input, value for powerbank.available. Available by default.

/deletepb

id select input, id of the powerbank to be removed. First available id by default.

/modifypb

id select input, value for powerbank to change by id.
name string input, new value for powerbank.name, may be left empty (prevents change).
brand string input, new value for powerbank.brand, may be left empty (prevents change).
battery_time number input, new value for powerbank.battery_time, may be left empty (prevents change).
charge_duration number input, new value for powerbank.charge_duration, may be left empty (prevents change).
cost number input, new value for powerbank.cost, may be left empty (prevents change).
available select input, new value for powerbank.available, may be left empty (prevents change).
